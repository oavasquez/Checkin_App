# Screenshot Checkin_App
Esto es un screenshoot de una aplicacion basada en el framework de flutter para aplicacion movil

La aplicacion tiene las funcionalidades basicas de un CheckStore conectada con Odoo como backend y el envio de la informacion para conectar ambos sistemas es usando xml rcp


## Inicio de sesion
Esta plataforma utiliza credenciales para el acceso exclusivo de los usuarios y ademas se puede configurar hacia el servidor que se desea dirigir las peticiones.
